# ♠ PocketAces Home Game Manager

> **Status:** Pending Apple Developer Account approval for App Store release.

A real-time iOS app for managing poker home games. Host a room, invite your crew, and let everyone log their own buy-ins and cash-outs as the night unfolds — no spreadsheets, no arguments at payout time.

## Features

- **Room Hosting** — Create a game session and share a room code with other players
- **Live Join & Log** — Players join rooms in real time and record their own buy-ins and cash-outs
- **Automatic Payouts** — Instant payout calculation at the end of the game; no mental math required
- **Individual Stats** — Track net profit/loss, session history, and performance trends per player
- **Real-Time Sync** — All data updates live across every player's device as the game progresses
